# Mini Pipeline Change Log

## Goal
Run an end-to-end validation pipeline on a 3% subset with minimal compute, without mutating full-data artifacts.

## Changes Applied
1. `TensorDataset/datsetSplitter.py`
- Added `HATEXPLAIN_SPLIT_FILE` override for split id source.
- Added `HATEXPLAIN_CACHE_SUFFIX` support so cached pickles for mini runs are isolated from full-data caches.
- Added backup/restore handling so mini cache generation does not overwrite the default full cache pickle.

2. `Models/utils.py`
- Added `HATEXPLAIN_DATA_FILE` and `HATEXPLAIN_CLASS_FILE` overrides inside `return_params`.

3. `manual_training_inference.py`
- Added `HATEXPLAIN_EPOCHS` override for easy low-compute runs.
- Added `HATEXPLAIN_DATA_FILE` and `HATEXPLAIN_CLASS_FILE` overrides in the main execution block.

4. `testing_with_rational.py`
- Added `HATEXPLAIN_SPLIT_FILE`, `HATEXPLAIN_DATA_FILE`, and `HATEXPLAIN_CLASS_FILE` overrides.
- Added `HATEXPLAIN_OUTPUT_SUFFIX` support to avoid output collisions.
- Updated sklearn `compute_class_weight` call to keyword-argument signature.

5. `testing_with_lime.py`
- Added `HATEXPLAIN_SPLIT_FILE`, `HATEXPLAIN_DATA_FILE`, and `HATEXPLAIN_CLASS_FILE` overrides.
- Added `HATEXPLAIN_OUTPUT_SUFFIX` support to avoid output collisions.
- Updated sklearn `compute_class_weight` call to keyword-argument signature.

6. `testing_for_bias.py`
- Added `HATEXPLAIN_SPLIT_FILE`, `HATEXPLAIN_DATA_FILE`, and `HATEXPLAIN_CLASS_FILE` overrides.
- Added `HATEXPLAIN_OUTPUT_SUFFIX` support to avoid output collisions.
- Updated sklearn `compute_class_weight` call to keyword-argument signature.

7. New script: `scripts/create_mini_subset.py`
- Generates sampled dataset and split files (default 3%).

8. New script: `scripts/build_eraser_eval_data.py`
- Builds ERASER-format `docs/`, `train.jsonl`, `val.jsonl`, and `test.jsonl` from a selected dataset/split pair.

## Commands Used
1. Build 3% subset
- `/bin/python3 scripts/create_mini_subset.py --ratio 0.03 --seed 42 --min-per-split 10`

2. Train 3-class mini model
- `HATEXPLAIN_DATA_FILE=Data/dataset_3pct.json HATEXPLAIN_SPLIT_FILE=Data/post_id_divisions_3pct.json HATEXPLAIN_CLASS_FILE=Data/classes.npy HATEXPLAIN_CACHE_SUFFIX=mini3pct_bert HATEXPLAIN_EPOCHS=1 conda run -n hatexplain37 python manual_training_inference.py best_model_json/bestModel_bert_base_uncased_Attn_train_TRUE.json True 0.003`

3. Generate rational outputs
- `HATEXPLAIN_DATA_FILE=Data/dataset_3pct.json HATEXPLAIN_SPLIT_FILE=Data/post_id_divisions_3pct.json HATEXPLAIN_CLASS_FILE=Data/classes.npy HATEXPLAIN_CACHE_SUFFIX=mini3pct_bert HATEXPLAIN_OUTPUT_SUFFIX=mini3pct conda run -n hatexplain37 python testing_with_rational.py bert_supervised 0.003`

4. Generate LIME outputs
- `HATEXPLAIN_DATA_FILE=Data/dataset_3pct.json HATEXPLAIN_SPLIT_FILE=Data/post_id_divisions_3pct.json HATEXPLAIN_CLASS_FILE=Data/classes.npy HATEXPLAIN_CACHE_SUFFIX=mini3pct_bert HATEXPLAIN_OUTPUT_SUFFIX=mini3pct conda run -n hatexplain37 python testing_with_lime.py bert_supervised 10 0.003`

5. Train 2-class mini model for bias
- `HATEXPLAIN_NUM_CLASSES=2 HATEXPLAIN_DATA_FILE=Data/dataset_3pct.json HATEXPLAIN_SPLIT_FILE=Data/post_id_divisions_3pct.json HATEXPLAIN_CLASS_FILE=Data/classes_two.npy HATEXPLAIN_CACHE_SUFFIX=mini3pct_bert_2cls HATEXPLAIN_EPOCHS=1 conda run -n hatexplain37 python manual_training_inference.py best_model_json/bestModel_bert_base_uncased_Attn_train_TRUE.json True 0.003`

6. Generate bias outputs
- `HATEXPLAIN_DATA_FILE=Data/dataset_3pct.json HATEXPLAIN_SPLIT_FILE=Data/post_id_divisions_3pct.json HATEXPLAIN_CLASS_FILE=Data/classes_two.npy HATEXPLAIN_CACHE_SUFFIX=mini3pct_bert_2cls HATEXPLAIN_OUTPUT_SUFFIX=mini3pct conda run -n hatexplain37 python testing_for_bias.py bert_supervised 0.003`

7. Build mini ERASER data
- `conda run -n hatexplain37 python scripts/build_eraser_eval_data.py --data-file Data/dataset_3pct.json --split-file Data/post_id_divisions_3pct.json --class-file Data/classes.npy --save-path Data/Evaluation/Model_Eval_3pct --bert-tokens true --max-length 128 --method union`

8. ERASER metrics
- `cd eraserbenchmark && PYTHONPATH=./:$PYTHONPATH conda run -n hatexplain37 python rationale_benchmark/metrics.py --split test --strict --data_dir ../Data/Evaluation/Model_Eval_3pct --results ../explanations_dicts/bestModel_bert_base_uncased_Attn_train_TRUE_0.003_explanation_top5_mini3pct.json --score_file ../model_explain_output_rational_mini3pct.json`
- `cd eraserbenchmark && PYTHONPATH=./:$PYTHONPATH conda run -n hatexplain37 python rationale_benchmark/metrics.py --split test --strict --data_dir ../Data/Evaluation/Model_Eval_3pct --results ../explanations_dicts/bestModel_bert_base_uncased_Attn_train_TRUE_explanation_with_lime_10_0.003_mini3pct.json --score_file ../model_explain_output_lime_mini3pct.json`
